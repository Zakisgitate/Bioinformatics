# Bioinformatics
# GSE204844_KDM6A_scRNA_pipeline

## Overview
This repository contains an R-based single-cell RNA-seq analysis pipeline for the **GEO dataset GSE204844**, with a specific focus on **KDM6A**, **inflammasome-related signaling** (`Nlrp3`, `Pycard`, `Casp1`, `Il1b`), and **NETosis/neutrophil-associated genes** (`Cybb`, `S100a8`, `S100a9`, `Camp`, `Ly6g`) in a mouse chronic pancreatitis context.

The pipeline was developed in **R / RStudio** and is compatible with **Seurat v5**. It performs raw matrix import, gene ID conversion, quality control, clustering, rough cell-type annotation, targeted myeloid/neutrophil analysis, and publication-oriented visualization.

---

## Main Objectives
This project was designed to answer the following questions:

1. Where is **Kdm6a** mainly expressed in the single-cell landscape?
2. Is **Kdm6a** spatially associated with **NLRP3 inflammasome-related genes**?
3. Are **NETosis-like inflammatory features** concentrated in a specific immune cell population?
4. How do **macrophage/myeloid cells** and **neutrophils** differ in the expression of inflammatory and neutrophil-associated signatures?

---

## Dataset
- **GEO accession:** GSE204844
- **Species:** *Mus musculus*
- **Input files used in this analysis:**
  - `GSM6198711_Case.counts.tsv`
  - `GSM6198712_Control.counts.tsv`

These files are treated as **single-cell count matrices**, where:
- rows = genes
- columns = cell barcodes
- values = raw counts

---

## Pipeline Summary
The workflow includes the following steps:

1. **Read raw count matrices**
2. **Convert mouse Ensembl IDs to gene symbols**
3. **Construct Seurat objects**
4. **Perform quality control and filtering**
5. **Merge Case and Control samples**
6. **Run standard scRNA-seq workflow**
   - normalization
   - variable feature selection
   - PCA
   - neighborhood graph construction
   - clustering
   - UMAP
7. **Explore target genes and module scores**
   - inflammasome-related module
   - NETosis-related module
8. **Perform marker analysis**
9. **Conduct rough cell-type annotation**
10. **Focus on macrophage/myeloid cells and neutrophils**
11. **Generate heatmaps and bubble plots**

---

## Key Findings from This Analysis
This exploratory analysis suggests that:

- **Kdm6a** is mainly enriched in **macrophage/myeloid cells**
- **Nlrp3**, **Pycard**, and **Il1b** are also primarily associated with the **macrophage/myeloid compartment**
- **Neutrophil-associated / NETosis-like genes** such as `Cybb`, `S100a8`, `S100a9`, `Camp`, and `Ly6g` are more strongly enriched in the **Case-associated neutrophil population**
- This pattern supports the idea that **Kdm6a may be more closely linked to macrophage-centered inflammasome signaling**, whereas **NETosis-like inflammatory features are mainly concentrated in neutrophils**

> Note: this analysis is **exploratory** because the current dataset includes only **one Case sample and one Control sample**, without biological replicates.
